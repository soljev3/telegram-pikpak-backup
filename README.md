### 第三步：綁定 PikPak 與 Telegram

2. 長按「通過 PikPak App 授權 Bot」按鈕，複製連結
3. 打開 PikPak 網頁版設定頁面
4. 將連結貼到「綁定 Telegram」文字框，點擊綁定

### 第四步：開始轉存！

綁定完成之後，你就可以直接：
